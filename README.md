# PHONE SCREEN

## Help

phone-screen

    Usage: phone-screen [-f FRAME] { -o OFILE SCREENSHOT | SCREENSHOTS... }
    
    Embed screenshot in a phone frame and output a SVG. When no "-o" is given
    then the output name is "SCREENSHOT.FRAME.png".
    
    Supported frames: x49tf
